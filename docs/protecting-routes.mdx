# Protecting routes

You may need to prevent access to some routes, for example if the user
does not have the rights to access them.

## Not found page

You can show the default "not found" page by returning `NotFound()` from a route map:

```dart
'/protected-route': (route) => 
    canUserAccessPage()
      ? MaterialPage(child: ProtectedPage())
      : NotFound()
```

## Redirection

Return `Redirect()` to redirect to another path. When running in a browser this will also update the URL.

```dart
'/protected-route': (route) => 
    canUserAccessPage()
      ? MaterialPage(child: ProtectedPage())
      : Redirect('/no-access'),
```

If you don't want the URL to change, simply return a different page:

```dart
'/protected-route': (route) => 
    canUserAccessPage()
      ? MaterialPage(child: ProtectedPage())
      : MaterialPage(child: CustomNoAccessPage())
```
